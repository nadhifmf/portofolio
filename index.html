E-Faskon OCR Database

E-Faskon OCR Database for Indonesian National Police.

Tech Stack
  • Python
  • Guillotina
  • On-Premise Linux Server
Features
  • PDF document storage
  • Automatic document OCR using on-premise API
  • PDF and content searching using OCR results

The backend side of E-Faskon is built using Python, powered by the Guillotina framework, and utilizes PostgreSQL for structured data management. BMKGSoftv2

BMKGSoft v2 Database for Indonesian Agency for Meteorological, Climatological and Geophysics.

Tech Stack
  • Python
  • Guillotina
  • Numpy and Pandas
Features
  • Generating PDF Report for Weather Report
  • Makes Weather and Climatological Statistics
  • Processing raw weather data

BMKGSoft v2 is a software solution designed for the storage of climate, climatological, and geophysical data, which is used to process raw data into a format ready for expert analysis. The system generates weather reports by applying predefined formulas using computational libraries such as NumPy, while leveraging Matplotlib for simple visualizations and Pandas for processing input CSV data.

ODSK Regional Public Hospital Dashboard

ODSK Regional Public Hospital Manado Data Dashboard.

Tech Stack
  • Python
  • Flask
  • On-Premise Linux Server
Features
  • Make SQL Query from raw database
  • Processing raw data to statistics
  • Notify user about patient growth

The database processing software for the ODSK Regional Public Hospital is built using Python Flask, enabling hospital administrators to access patient statistics based on payment methods such as BPJS (social security) or patient categories like hemodialysis treatments. This system allows hospital administrators to monitor hospital conditions without the need for manual SQL queries or reliance on Excel for data analysis.
